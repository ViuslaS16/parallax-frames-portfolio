## PROJECT IDENTITY

Build a premium immersive web experience for an EDM event photographer.

IMPORTANT:

This is NOT a traditional photographer portfolio.

This is:
> an interactive archive of nightlife culture.

The website should feel like:
- entering an underground music festival
- reliving moments from a live set
- browsing emotional memories from nightlife culture
- a cinematic digital exhibition

The emotional atmosphere is more important than conventional portfolio structure.

---

# TECH STACK

Use:

- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS
- Framer Motion
- Lenis smooth scrolling
- Sanity CMS integration ready
- Fully responsive mobile-first architecture
- SEO optimized

---

# MOTION SYSTEM

Framer Motion animations are CRITICAL.

Animations should feel:
- cinematic
- restrained
- elegant
- rhythmic
- emotionally paced

DO NOT create flashy crypto-style animations.

The website pacing should mimic EDM structure:
- anticipation
- buildup
- release
- silence
- climax

---

# FRAMER MOTION REQUIREMENTS

## 1. Shared Layout Transitions

Use `layoutId` transitions between:
- gallery thumbnails
- event pages
- hero images

Images should seamlessly expand into fullscreen cinematic event pages.

---

## 2. Staggered Reveal Animations

Gallery images should reveal progressively using stagger timing.

Use:
- delayed sequencing
- directional reveals
- layered pacing

---

## 3. Cinematic Hero Entrances

Large typography and hero images should:
- fade in
- sharpen from blur
- slightly translate vertically
- animate slowly and smoothly

Motion must feel cinematic and luxurious.

---

## 4. Scroll-Based Motion

Use scroll transforms for:
- image scaling
- opacity changes
- subtle parallax
- atmospheric pacing

The website should feel musically paced.

---

## 5. Hover Interactions

Use minimal premium hover interactions:
- subtle scale
- ambient glow
- soft depth movement
- magnetic feeling

Hover motion should feel expensive and restrained.

---

## 6. Page Transitions

All route transitions must feel seamless.

No hard cuts.

Use:
- fade layering
- image continuity
- motion continuity

---

## 7. Backstage Mode Transition

When entering the “Backstage” section:
- transition from black UI to white UI
- motion becomes sharper and more technical
- atmosphere shifts from emotional to operational

This should feel like:
> leaving the dancefloor and entering production control.
